Diseño y programación de sitios Web.

Universidad Nacional Abierta y a Distancia.

Objetivo general.

El Objetivo de este curso es implementar a los estudiantes las generalidades del diseño web para poder utilizar de forma debida los lenguajes HTML5 y CSS3. para que puedan aplicarlos en la construcción de páginas web a través de sistemas de control de versiones.

Objetivo específico 1.

Ofrecer informacón al estudiante, para que este pueda operar y administrar los sistemas de control de versiones tales como GIT o GITHUB.

Objetivo específico 2.

Explicar la estructura básica de HTML5 (etiquetas, atributos y valores), la estructura básica de páginas (HTML, HEAD, BODY, FOOTER).

Objetivo específico 3.

Brindar las pautas necesarias para que el estudiante evalúe y desarrolle sitios web que puedan satisfacer las necesidades del usuario y dar recomendaciones para su actualización.
